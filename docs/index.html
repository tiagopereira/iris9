Overview

Here you can find the materials for the IRIS data analysis tutorial at the IRIS-9 Workshop. This tutorial session is interactive and participants are expected to work from their laptops. During the first part there will be question sessions.

To answer the questions the participants are encouraged to consult the IRIS documentation and resources, in particular the following:

  • Guide to IRIS data analysis (ITN 26)
  • IRIS instrument paper
  • IRIS data search
  • IRIS today
